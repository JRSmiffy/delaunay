## Explanation
- 🧮For a set of points `P`:
    - The Delaunay Triangulation `DT(P)`, is such that no point in `P` lies inside the circumcircle of any triangle in `DT(P)`
- ℹ️ [More info](https://en.wikipedia.org/wiki/Delaunay_triangulation)
